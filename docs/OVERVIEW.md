# Overview

The **League Standings Leaderboard & Live Performance Analytics Portal** is a
broadcast-style React + TypeScript app that runs a **simulated** mini-league
between four **real clubs** (Real Madrid, FC Barcelona, Manchester United, Bayern
Munich — real crests, official kit colours, and current first-team squads
snapshotted 2026-06-29) and turns it into live performance analytics. Its
defining choice is a strict single-source-of-truth architecture: the whole season
is generated by a seeded simulation engine (mulberry32 + Poisson goal draws on
rating-derived xG), and every view — table, stats, MOTM, form, win-probability,
clinch, leaderboards, momentum, the position worm and the analyst's
predictions — is a pure function of the resulting match-log, so nothing can
desync and a **What-If** edit recomputes the entire cascade and FLIPs the table.
The interactive layer makes the seed mutable: **Reset Season** cycles ~12 curated
dramatic seasons, **Surprise me** rolls a random one, and a scrubber replays the
campaign both ways. The finish is a restrained broadcast skin on an iconic palette
(ink surfaces, electric-cyan as the only brand accent, champion gold reserved for
the leader/champion, win/loss greens and reds): a FIFA-style momentum wave, a
tabbed Stats/Tables/Match/Season area beneath an always-primary hero, El Clásico
framing on the rivalry fixture, and broadcast moments (whistle and crowd SFX, a
gold CHAMPIONS confetti moment on clinch). The creative centrepiece is a two-way
AI Performance Analyst — a deep tactical read, free-form Q&A grounded only in the
live data, and a season-long prediction track record — powered by a direct
client-side Gemini (`gemini-2.5-flash`) call with an in-memory key and a
deterministic grounded fallback, always framed as a simulation. `npm run verify`
proves the engine and analytics deterministically, including across 20 random
seeds and What-If edits.
